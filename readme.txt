Ultimate Product Options For WooCommerce allows you to add custom fields and extra options to WooCommerce products effortlessly.

== Description ==

Ultimate Product Options For WooCommerce provides an easy way to add custom fields and extra options to your WooCommerce products. Enhance your product listings by offering more customization options for your customers, helping you increase sales and improve user satisfaction.

= Features =

* Add custom fields to WooCommerce products
* Display extra options in product pages
* Seamlessly integrates with WooCommerce
* Easy to configure and use
* Supports various field types such as text, checkbox, radio buttons, and more

= Installation =

1. Upload the plugin files to the `/wp-content/plugins/ultimate-product-options-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

= Frequently Asked Questions =

= How do I add custom fields to my products? =

After activating the plugin, go to WooCommerce  > Product Extra Data and configure your custom fields. You can choose the field type and specify the options for each field.

= Can I display custom fields in the WooCommerce cart and checkout pages? =

Yes, the custom fields added to products will be displayed in the WooCommerce cart and checkout pages, allowing customers to see their selected options.

= Does the plugin support conditional logic for fields? =

Currently, the plugin does not support conditional logic for fields. We are working on adding this feature in future updates.

= Can I customize the appearance of the custom fields? =

Yes, you can customize the appearance of the custom fields using CSS. The plugin provides classes for each field type, allowing you to style them according to your theme.
